# ADR-012: First useful output is coordination artifact, not itinerary

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** NEW
- **Source:** @Today : release; Agentic Hackathon
- **Notion URL:** https://app.notion.com/p/8aa881a7c09d46efad13672dc46781b4

## Context & Options

Coordination failure happens before planning converges. Full itinerary output is too broad for the first loop.

## Decision

The first agent output should be a poll, candidate weekends, or a current-plan summary, not an itinerary or booking recommendation.

## Consequences

The demo remains narrow and directly addresses unpaid group coordination work.
