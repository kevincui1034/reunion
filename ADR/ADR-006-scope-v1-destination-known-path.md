# ADR-006: Scope V1 to the destination-known planning path

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** NEW
- **Source:** Hackathon Planning Discussion; Hackathon work session
- **Notion URL:** https://app.notion.com/p/63ef45c637954db5bcf85beaa40dc7fc

## Context & Options

Three path types were identified: destination-known, time-known, and open-ended. Supporting all three in V1 would spread scope too thin.

## Decision

V1 handles destination-known first: known where -> determine who -> determine when.

## Consequences

The demo path is clearer and deliverable within hackathon constraints.
