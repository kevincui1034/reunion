# ADR-009: Treat group travel planning as coordination memory

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** NEW
- **Source:** Today : release; Agentic Hackathon; Group Travel Planning Agent
- **Notion URL:** https://app.notion.com/p/af89f467e6c64305a2b85510524afd47

## Context & Options

Core pain is buried intent and constraints across conversations and people, not just travel search.

## Decision

Frame the product as a coordination-memory agent rather than a generic travel assistant.

## Consequences

XTrace becomes load-bearing and outputs must leverage remembered constraints.
