# ADR-001: Use Reunion as the working repository name

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** NEW
- **Source:** Hackathon work session
- **Notion URL:** https://app.notion.com/p/b2b8f5f4b33e45f5b09b90eed03bc253

## Context & Options

The team needed a repository name immediately. "Reunion" fit the social planning theme and avoided spending hackathon time on branding.

## Decision

Use "Reunion" as the repository/project name for now and defer final branding.

## Consequences

GitHub setup and docs can proceed immediately. The name remains easy to change later, and branding does not block prototype progress.
