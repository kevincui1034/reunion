# ADR-010: Use WhatsApp-first as the primary surface (refines prior decision)

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** REFINES
- **Source:** Today : release; Agentic Hackathon; Group Travel Planning Agent; prior WhatsApp-first ADR
- **Notion URL:** https://app.notion.com/p/ba4bcbdb94034c09a470f73b8563b60c

## Context & Options

The team considered iMessage, WhatsApp, Discord, and standalone app options. WhatsApp best balanced user fit and implementation feasibility.

## Decision

Use WhatsApp-first group chat as the primary product surface for the hackathon prototype.

## Consequences

The demo runs where the user problem already exists, while richer standalone UX is deferred.
