# ADR-018: Start with the destination-known travel path

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** NEW
- **Source:** @Today : release; Agentic Hackathon
- **Notion URL:** https://app.notion.com/p/29b89123abb545019981e50d0b33eb5e

## Context & Options

The team identified destination-known, time-known, and open-ended paths. Building all paths in hackathon scope adds too much risk.

## Decision

For the hackathon prototype, support the destination-known path first: known where, then determine who can go and when.

## Consequences

The demo gains one crisp, shippable flow while other paths remain future work.
