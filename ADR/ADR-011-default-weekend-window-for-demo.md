# ADR-011: Default trip planning to a weekend window for demo

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** NEW
- **Source:** Agentic Hackathon
- **Notion URL:** https://app.notion.com/p/d8353b7569874f3bbda7277c7d1c8dfb

## Context & Options

Trip length is a hidden degree of freedom that complicates candidate-date generation.

## Decision

Default candidate trips to a weekend window: Friday midday departure through Monday return.

## Consequences

Date-generation logic stays constrained and demo-friendly; custom trip lengths are future work.
