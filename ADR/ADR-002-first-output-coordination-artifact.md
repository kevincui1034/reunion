# ADR-002: Make the first output a coordination artifact

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** NEW
- **Source:** Today : release; Hackathon Planning Discussion
- **Notion URL:** https://app.notion.com/p/8ad0df28ce04479394c108e489b6edfe

## Context & Options

Full itinerary generation is downstream of group commitment. The smallest useful artifact is the next coordination move.

## Decision

Make the first useful output a poll, candidate-weekend list, or current-plan summary instead of a full itinerary.

## Consequences

The agent proves utility earlier and keeps scope narrow. Deep search/recommendation depth is deferred.
