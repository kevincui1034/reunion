# ADR-004: Use RocketRide as the visible workflow layer

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** NEW
- **Source:** Today : release; Agentic Hackathon; Group Travel Planning Agent
- **Notion URL:** https://app.notion.com/p/4e098b11b20f4bf49b8b6061496238e5

## Context & Options

Hackathon requirements call for meaningful RocketRide usage. A visible pipeline is more explainable than a one-shot LLM call.

## Decision

Use RocketRide as the visible agentic workflow layer instead of an opaque prompt.

## Consequences

The demo shows real orchestration and debugging visibility, with added integration overhead.
