# ADR-008: Use mocked 12-month calendar data for the demo

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** NEW
- **Source:** Hackathon Planning Discussion; Hackathon work session
- **Notion URL:** https://app.notion.com/p/889feeda043e497894ae20ee522836a6

## Context & Options

Live multi-user OAuth and consent create delivery risk during hackathon timelines.

## Decision

Use mocked or demo-account calendar data for 12 months across four people, with real Google Calendar integration as the production path.

## Consequences

The demo can reliably show coordination logic, while production auth is deferred.
