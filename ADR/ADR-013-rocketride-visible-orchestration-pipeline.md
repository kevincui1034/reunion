# ADR-013: Make RocketRide the visible orchestration pipeline

- **Status:** Accepted
- **Date:** 2026-06-05
- **Reconciliation:** NEW
- **Source:** @Today : release; Group Travel Planning Agent
- **Notion URL:** https://app.notion.com/p/72f592d35a9a423589b418d5910b3c88

## Context & Options

The hackathon requires meaningful RocketRide usage. Workflow steps should be inspectable during demos.

## Decision

Use RocketRide as the visible orchestration layer for intent classification, extraction, routing, memory-aware planning, and next-action generation.

## Consequences

Judges can see a real pipeline, with added complexity versus one-shot prompting.
